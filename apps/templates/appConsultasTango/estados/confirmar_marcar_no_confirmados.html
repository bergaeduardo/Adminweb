Confirmación de Operación

¿Qué hace esta operación?

Esta operación buscará todos los turnos que cumplan las siguientes condiciones:

  • Estado actual: RESERVADO
  • Horario programado hace más de 30 minutos
  • No fueron confirmados a tiempo

Esos turnos serán marcados automáticamente como NO CONFIRMADO.

Advertencia

Esta acción cambiará el estado de los turnos identificados y registrará el cambio en el historial como ejecutado por el usuario SISTEMA. Esta operación no se puede deshacer automáticamente.

Información Adicional

¿Cuándo se debe ejecutar este proceso?

Se recomienda ejecutar este proceso en los siguientes casos:

  • Periódicamente durante el día (cada 30-60 minutos)
  • Al final del día para limpiar turnos no confirmados
  • Manualmente cuando se detecten turnos que no se confirmaron a tiempo
Automatización

Para automatizar este proceso, puede configurar una tarea programada de Windows que ejecute el script correspondiente, o contactar con el administrador del sistema para configurar una tarea programada en el servidor.

Estados que no bloquean horarios

Los siguientes estados liberan el horario para nuevas reservas:

  • CANCELADO
  • RECHAZADO
  • NO CONFIRMADO
